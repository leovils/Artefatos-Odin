Passo 0 — Quando Usar Testes Não-Paramétricos?
🧙‍♂️ Odin Dinho explica:
Nem sempre os dados seguem distribuição normal. Amostras pequenas, distribuições assimétricas, dados ordinais — quando os pressupostos do teste t falham, precisamos de ferramentas diferentes. Os testes não-paramétricos são essas ferramentas.
⚡ ThorMento acha:
"Bah, teste t resolve tudo! Joga os dados lá e pronto!"
🧙‍♂️ Odin Dinho responde:
Não quando os pressupostos são violados, ThorMento. Com n = 7 ou 8, não dá para garantir normalidade. O teste t pode dar resultados enganosos.
🐦‍⬛ Muninn sussurra:
Testes não-paramétricos trabalham com RANKINGS, não com valores brutos. Isso os torna robustos contra outliers e distribuições estranhas. A lógica é simples: se um grupo tende a ter ranks maiores, ele tende a ter valores maiores.

Passo 2 — Hipóteses do Mann-Whitney U
🧙‍♂️ Odin Dinho explica:
O Mann-Whitney U testa se as distribuições dos dois grupos são iguais. Não é apenas sobre médias — é sobre a tendência geral de um grupo ter valores maiores que o outro.
H₀ (Hipótese Nula)
As distribuições dos dois grupos são iguais
P(X > Y) = 0,5
H₁ (Hipótese Alternativa)
As distribuições diferem
P(X > Y) ≠ 0,5
🐦‍⬛ Muninn sussurra:
Note que H₀ é sobre as DISTRIBUIÇÕES, não apenas as médias. Se as formas das distribuições forem iguais, aí sim podemos interpretar como diferença de locação (medianas).
🧙‍♂️ Odin Dinho explica:
A lógica é simples: se H₀ for verdadeira, os ranks devem se distribuir igualmente entre os clãs. Se um clã acumular ranks mais altos, temos evidência contra H₀.
⚡ ThorMento diz:
"Ranks? Que perda de tempo! Os valores originais são mais informativos!"
🧙‍♂️ Odin Dinho responde:
Os ranks nos protegem de outliers, ThorMento. Se um mercador trocasse 250 peças em vez de 25, o rank dele seria o mesmo — 15. O teste t seria distorcido, o Mann-Whitney não.

Passo 4 — Cálculo da Estatística U
🧙‍♂️ Odin Dinho explica:
A estatística U transforma as somas de ranks em uma medida de sobreposição entre os grupos. Quanto menor o U, maior a separação.
Fórmula de U₁
U₁ = n₁ × n₂ + n₁(n₁ + 1) / 2 − R₁
Cálculo Passo a Passo: U₁
n₁ × n₂8 × 756
n₁(n₁ + 1) / 28 × 9 / 236
R₁77
U₁ = 56 + 36 − 7715
Cálculo Passo a Passo: U₂
n₁ × n₂8 × 756
n₂(n₂ + 1) / 27 × 8 / 228
R₂43
U₂ = 56 + 28 − 4341
Verificação
U₁ + U₂ = 15 + 41 = 56 = n₁ × n₂ = 8 × 7 = 56 ✓
Estatística U
U = min(U₁, U₂) = min(15, 41) = 15

Passo 5 — Significância Estatística
🧙‍♂️ Odin Dinho explica:
Com amostras pequenas, podemos usar tabelas de valores críticos. Para amostras maiores (n > 20), usamos a aproximação pela normal (z). Vou mostrar as duas abordagens.
Abordagem 1: Tabela de Valores Críticos
Para n₁ = 8, n₂ = 7, α = 0,05 (bilateral)
Ucrítico = 13

Rejeitamos H₀ se U ≤ Ucrítico

Nosso U = 15 > 13 → Não rejeitamos H₀

Abordagem 2: Aproximação Normal (z)
Média esperada de U sob H₀
μU = n₁ × n₂ / 2 = 8 × 7 / 2 = 28
Desvio padrão de U
σU = √(n₁ × n₂ × (n₁ + n₂ + 1) / 12) = √(8 × 7 × 16 / 12) = √(74,667) = 8,641
Escore z
z = (U − μU) / σU = (15 − 28) / 8,641 = −13 / 8,641 = −1,505
p-valor (bilateral)
p = 2 × P(Z ≤ −1,505) ≈ 2 × 0,0662 = 0,1324
Decisão
p = 0,132 > α = 0,05 → Não rejeitamos H₀
Tamanho de Efeito
r (correlação de rank bisserial)
r = |z| / √N = 1,505 / √15 = 1,505 / 3,873 = 0,389
Efeito médio (Cohen: 0,1 = pequeno, 0,3 = médio, 0,5 = grande)
🐦‍⬛ Muninn sussurra:
Mesmo não sendo significativo, o tamanho de efeito r = 0,389 sugere uma diferença prática moderada. P-value e effect size contam histórias diferentes!

Passo 8 — Conceito do Wilcoxon Signed-Rank
🧙‍♂️ Odin Dinho explica:
O Wilcoxon trabalha com as diferenças entre cada par. A lógica tem quatro passos claros:
Algoritmo do Wilcoxon Signed-Rank
  1. Calcular D = Depois − Antes para cada par
  2. Descartar pares com D = 0
  3. Ranquear os |D| (valores absolutos)
  4. Somar ranks positivos (T⁺) e negativos (T⁻) separadamente
🐦‍⬛ Muninn sussurra:
A sacada: se o tratamento não tem efeito, as diferenças positivas e negativas devem ser equilibradas — T⁺ ≈ T⁻. Se o tratamento funciona, um deles dominará.
Hipóteses
H₀ (Hipótese Nula)
A distribuição das diferenças é simétrica em torno de zero
Mediana das diferenças = 0
H₁ (Hipótese Alternativa)
A distribuição das diferenças NÃO é simétrica em torno de zero
Mediana das diferenças ≠ 0

Passo 12 — Significância do Wilcoxon
Abordagem 1: Tabela de Valores Críticos
Para n = 8 pares, α = 0,05 (bilateral)
Tcrítico = 3

Rejeitamos H₀ se T ≤ Tcrítico

Nosso T = 1 ≤ 3 → Rejeitamos H₀!

Abordagem 2: Aproximação Normal (z)
Média esperada de T sob H₀
μT = n(n+1) / 4 = 8 × 9 / 4 = 18
Desvio padrão de T
σT = √(n(n+1)(2n+1) / 24) = √(8 × 9 × 17 / 24) = √(51) = 7,141
Escore z
z = (T − μT) / σT = (1 − 18) / 7,141 = −17 / 7,141 = −2,380
p-valor (bilateral)
p = 2 × P(Z ≤ −2,380) ≈ 2 × 0,00866 = 0,0173
Tamanho de Efeito
r (effect size)
r = |z| / √(2n) = 2,380 / √16 = 2,380 / 4 = 0,595
Efeito grande (Cohen: 0,1 = pequeno, 0,3 = médio, 0,5 = grande)

Passo 14 — Lição dos Corvos

LIÇÃO DOS CORVOS

🐦‍⬛ Muninn resume:
Pontos-chave que você deve guardar:

1. Testes não-paramétricos trabalham com RANKS, não valores brutos — isso os protege de outliers e distribuições estranhas.

2. Mann-Whitney U: para grupos independentes. Combina todos os dados, ranqueia, soma ranks por grupo. U mede sobreposição.

3. Wilcoxon Signed-Rank: para dados pareados. Calcula diferenças, ranqueia |D|, soma ranks por sinal. T mede assimetria das diferenças.

4. Sempre verifique: R₁ + R₂ = N(N+1)/2 para Mann-Whitney e T⁺ + T⁻ = n(n+1)/2 para Wilcoxon.

5. Reporte em APA: estatística, z, p-valor E tamanho de efeito (r).
🐦‍⬛ Huginn desafia:
Perguntas para reflexão:

1. E se os dados fossem ordinais (escala Likert)? Os testes não-paramétricos continuam válidos? (Sim — ranks funcionam naturalmente com dados ordinais.)

2. O tamanho de efeito r tem a mesma interpretação que o r de Pearson? (Não exatamente — são baseados em ranks, mas as convenções de Cohen se aplicam.)

3. Quando usar a tabela de valores críticos vs a aproximação z? (Tabela para n ≤ 20; z para amostras maiores. A tabela é exata; z é aproximação.)

4. Existe versão não-paramétrica para mais de 2 grupos? (Sim! Kruskal-Wallis para independentes, Friedman para pareados.)
🧙‍♂️ Odin Dinho conclui:
Não-paramétricos não são "testes de segunda classe". São ferramentas essenciais quando os dados não cooperam — e isso acontece com frequência no mundo real. Um guerreiro sábio conhece todas as armas do arsenal, não apenas as favoritas.
